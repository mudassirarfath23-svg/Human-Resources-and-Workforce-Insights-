# Human Resources Trends and Workforce Insights

## Project Overview
This project focuses on analyzing Human Resources data to identify workforce trends and generate meaningful business insights. SQL was used for data extraction and analysis, while Microsoft Power BI was used to create interactive dashboards and visual reports.

## Objectives
- Analyze employee workforce data.
- Identify department-wise workforce trends.
- Generate HR insights through data visualization.
- Support data-driven decision making.

## Tools & Technologies
- Microsoft Power BI
- SQL
- Microsoft Excel
- Data Analytics

## Project Files
- HR_Dashboard.pbix – Power BI dashboard
- SQL_Queries.sql – SQL queries used for analysis
- Project_Report.pdf – Complete project documentation
- Output_Observed.pdf – Dashboard outputs and observations
- Database_Connection.txt – Database connection details

## Key Features
- Interactive HR Dashboard
- Workforce Trend Analysis
- Employee Data Insights
- SQL-based Data Analysis
- Business Intelligence Reporting

## Conclusion
This project demonstrates practical knowledge of SQL, Power BI, and data analysis by transforming HR data into meaningful visual insights for better decision-making
